Connexion Google Sheets

Collez les liens CSV publiés de vos 4 onglets. L'écran lira les données à chaque clic sur « Actualiser ». Tant qu'aucun lien n'est saisi, les données de démonstration (semaine 21) sont affichées.

Comment publier votre Google Sheets :
  1. Importez votre Excel dans Google Sheets
  2. Menu Fichier → Partager → Publier sur le web
  3. Pour chaque onglet, choisissez le format CSV puis cliquez sur Publier
  4. Copiez chaque lien généré dans le champ correspondant ci-dessus
  5. Gardez l'ordre des colonnes identique à l'Excel d'origine
Fonctionne en double-clic : ce dashboard sait récupérer les données même ouvert localement. Si la connexion directe est bloquée, il passe automatiquement par un relais. Le premier chargement peut prendre quelques secondes — c'est normal.
